SimME - @berlios

What does this stand for? Simulator MicroEdition. In fact this project develops a two player game for Java 2 MicroEdition. The game is based on a strategy game which can be found at www.dbai.tuwien.ac.at/proj/ramsey (the simulator). The game will be played via mobile phone against any other person around the globe.

A community page will be set up, providing rankings, stats and a possibility to match up. More on this soon.

Current Development

We are currently working hard on our next release. For the curious: have a look at the project's CVS web interface. Although some comments are in German, we tried to keep all javadoc in English.

The project is split into two parts. One part builds the client side which uses the MIDP-1.0 and CLDC-1.0 specification. The other part develops the server side which uses JSPs/Servlets (where we don't have that much experience). The two parts communicate via XML over HTTP.

Currently the development process is held very close but the project will soon be opened after we have created something someone else could use. There is a short introduction on the development environment. Although it is not current, there are some links that can get you started.

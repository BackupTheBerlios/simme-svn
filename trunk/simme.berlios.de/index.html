SimME - @berlios

What does this stand for? Simulator MicroEdition. In fact this project develops a two player game for Java 2 MicroEdition. The game is based on a strategy game which can be found at www.dbai.tuwien.ac.at/proj/ramsey (the simulator). The game will be played via mobile phone against any other person around the globe.

We are also trying to set up a web page with rankings, stats and a possibility to match up. More on this soon.

General Information

This is the main web page with some current information on the project. The most current version is 0.01 and is currently not accessible via the project's main page. You can download this development version at:

www.einspiel.at

Note: the above URL is a WAP page and may be displayed incorrectly in some browsers.

The files are intended for MIDP-1.0 and CLDC-1.0 specifications. More about that at java.sun.com/j2me or wireless.java.sun.com. The packages have been created with the wireless toolkit provided by Sun.

Current Development

We are currently working hard on our next release. For the curious: have a look at the project's CVS web interface. Although some comments are in German, we tried to keep all javadoc in English.

The project is split into two parts. One part builds the client side which is uses the MIDP-1.0 and CLDC-1.0 specification. The other part develops the server side which uses JSPs (we don't have any experience in using JSPs). The two parts communicate via XML over HTTP.

Currently the development process is held very close but the project will soon be opened after we have created something someone else could build up upon.
